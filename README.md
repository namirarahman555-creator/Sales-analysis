# Sales-analysis
Beginner sales data analysis project using Python, Pandas, and Matplotlib to analyze revenue, product sales, and city-wise performance.

## Overview
This project analyzes sales data using Python and Pandas. The objective is to gain insights into sales performance, identify best-selling products, and determine the city with the highest sales. Basic visualizations are created using Matplotlib.

Features
Calculate total revenue
Identify the best-selling product
Find the city with the highest sales
Visualize city-wise sales using bar graphs
Technologies Used
Python
Pandas
Matplotlib
Dataset

The dataset contains the following columns:

Order_ID
Product
Quantity
Price
City
Order_Date
